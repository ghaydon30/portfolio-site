The Meet App

The Meet App is a serverless web application built to enhance event discovery by integrating with the Google Calendar API. Users can filter events by city and explore graphical representations of event distributions through interactive charts. Developed with AWS Lambda, the app emphasizes modern development practices, including rigorous testing with Jest and Puppeteer, and a robust CI/CD pipeline for consistent deployment. Designed to practice advanced skills, the Meet App demonstrates expertise in testing methodologies, serverless architecture, and seamless API integration.
